# NewLife.NovaDb - 服务器架构

## 1. 概述

NovaServer 提供远程访问能力,基于 NewLife.Remoting.ApiServer 构建,支持 RPC 协议访问数据库,实现嵌入式与网络双模部署。

## 2. 架构组件

```
┌─────────────────────────────────────┐
│        NovaServer (入口)            │
│  - 端口监听                         │
│  - 生命周期管理                     │
│  - SqlEngine 实例（静态共享）       │
└──────────────┬──────────────────────┘
               ↓
┌─────────────────────────────────────┐
│   NovaController (RPC 控制器)       │
│  - Nova/Ping                        │
│  - Nova/Execute                     │
│  - Nova/Query                       │
│  - Nova/BeginTransaction            │
│  - Nova/CommitTransaction           │
│  - Nova/RollbackTransaction         │
└──────────────┬──────────────────────┘
               ↓
┌─────────────────────────────────────┐
│      SqlEngine (共享实例)           │
│  - SQL 解析与执行                   │
│  - 事务管理                         │
│  - 引擎调度 (Nova/Flux/KV)          │
└─────────────────────────────────────┘
```

### 2.1 NovaServer

服务器封装类,管理端口、生命周期和 SqlEngine 实例:

```csharp
public class NovaServer : IDisposable
{
    private readonly ApiServer _apiServer;
    private static SqlEngine? _sqlEngine; // 静态共享
    
    public NovaServer(ServerDbOptions options)
    {
        // 初始化 SqlEngine（全局单例）
        _sqlEngine ??= new SqlEngine(options);
        
        // 初始化 ApiServer
        _apiServer = new ApiServer(options.Port)
        {
            Name = "NovaServer",
            Log = XTrace.Log
        };
        
        // 注册控制器
        _apiServer.Register<NovaController>();
    }
    
    public void Start()
    {
        _apiServer.Start();
        XTrace.WriteLine($"NovaServer started on port {_apiServer.Port}");
    }
    
    public void Stop()
    {
        _apiServer.Stop();
        _sqlEngine?.Dispose();
    }
    
    public static SqlEngine GetEngine() => _sqlEngine 
        ?? throw new InvalidOperationException("SqlEngine not initialized");
}
```

### 2.2 NovaController

RPC 控制器,处理客户端请求:

```csharp
[Api("Nova")]
public class NovaController : IApi
{
    /// <summary>心跳,返回服务器时间</summary>
    public DateTime Ping()
    {
        return DateTime.Now;
    }
    
    /// <summary>执行 SQL（非查询）,返回影响行数</summary>
    public Int32 Execute(String sql, Dictionary<String, Object?>? parameters = null)
    {
        var engine = NovaServer.GetEngine();
        return engine.Execute(sql, parameters);
    }
    
    /// <summary>查询,返回列名和行数据</summary>
    public QueryResult Query(String sql, Dictionary<String, Object?>? parameters = null)
    {
        var engine = NovaServer.GetEngine();
        var rows = engine.Query(sql, parameters);
        
        // 提取列名
        var columns = rows.Count > 0 
            ? rows[0].Keys.ToList() 
            : new List<String>();
        
        return new QueryResult
        {
            Columns = columns,
            Rows = rows
        };
    }
    
    /// <summary>开始事务,返回事务 ID</summary>
    public Int64 BeginTransaction()
    {
        var engine = NovaServer.GetEngine();
        var tx = engine.BeginTransaction();
        return tx.TxId;
    }
    
    /// <summary>提交事务</summary>
    public void CommitTransaction(Int64 txId)
    {
        var engine = NovaServer.GetEngine();
        engine.CommitTransaction(txId);
    }
    
    /// <summary>回滚事务</summary>
    public void RollbackTransaction(Int64 txId)
    {
        var engine = NovaServer.GetEngine();
        engine.RollbackTransaction(txId);
    }
}

public class QueryResult
{
    public List<String> Columns { get; set; } = [];
    public List<Dictionary<String, Object?>> Rows { get; set; } = [];
}
```

## 3. 协议格式

### 3.1 固定头部（16 字节）

```
+--------+--------+--------+--------+--------+--------+--------+--------+
| Magic  | Version| ReqType| SeqId (4 bytes)  | PayloadLen (4B)         |
| (2B)   | (1B)   | (1B)   |                 |                         |
+--------+--------+--------+--------+--------+--------+--------+--------+
| Status | Reserved (3 bytes)                                          |
| (1B)   |                                                              |
+--------+--------+--------+--------+--------+--------+--------+--------+
```

| 字段 | 大小 | 说明 |
|------|------|------|
| Magic | 2 字节 | 魔数 `0x4E56`（"NV"） |
| Version | 1 字节 | 协议版本（当前为 1） |
| RequestType | 1 字节 | 请求类型（0=Ping, 1=Execute, 2=Query, 3=BeginTx, 4=CommitTx, 5=RollbackTx） |
| SequenceId | 4 字节 | 序列号,用于请求/响应匹配 |
| PayloadLength | 4 字节 | 载荷长度（字节） |
| Status | 1 字节 | 状态码（0=成功, 1=错误） |
| Reserved | 3 字节 | 保留字段 |

### 3.2 载荷格式

使用 JSON 序列化（未来可扩展为 Protobuf）:

```json
// Execute 请求
{
  "sql": "INSERT INTO users (id, name) VALUES (1, 'Alice')",
  "parameters": {
    "@id": 1,
    "@name": "Alice"
  }
}

// Execute 响应
{
  "affectedRows": 1
}

// Query 请求
{
  "sql": "SELECT * FROM users WHERE age > @minAge",
  "parameters": {
    "@minAge": 18
  }
}

// Query 响应
{
  "columns": ["id", "name", "age"],
  "rows": [
    {"id": 1, "name": "Alice", "age": 25},
    {"id": 2, "name": "Bob", "age": 30}
  ]
}
```

## 5. 性能优化

### 5.1 连接池

客户端维护连接池,复用连接:

```csharp
public class NovaConnectionPool
{
    private readonly ConcurrentBag<NovaConnection> _pool = [];
    
    public NovaConnection Acquire()
    {
        if (_pool.TryTake(out var conn) && conn.IsAlive)
        {
            return conn;
        }
        
        return CreateConnection();
    }
    
    public void Release(NovaConnection conn)
    {
        if (_pool.Count < MaxPoolSize)
        {
            _pool.Add(conn);
        }
        else
        {
            conn.Dispose();
        }
    }
}
```

### 5.2 批量执行

```csharp
/// <summary>批量执行 SQL</summary>
public List<Int32> BatchExecute(List<String> sqlList)
{
    var engine = NovaServer.GetEngine();
    var results = new List<Int32>();
    
    using var tx = engine.BeginTransaction();
    
    foreach (var sql in sqlList)
    {
        results.Add(engine.Execute(sql, tx));
    }
    
    tx.Commit();
    return results;
}
```

### 5.3 结果集流式传输

对于大结果集,采用流式传输避免内存溢出:

```csharp
/// <summary>流式查询（分批返回）</summary>
public IEnumerable<Dictionary<String, Object?>> StreamQuery(String sql, Int32 batchSize = 1000)
{
    var engine = NovaServer.GetEngine();
    var rows = engine.Query(sql);
    
    for (var i = 0; i < rows.Count; i += batchSize)
    {
        foreach (var row in rows.Skip(i).Take(batchSize))
        {
            yield return row;
        }
    }
}
```

## 7. 设计决策

### D1: 静态共享 SqlEngine
- 网络模式单进程运行,所有 RPC 请求共享同一个 SqlEngine 实例
- 简化资源管理,避免多实例竞争

### D2: 基于 NewLife.Remoting
- 复用成熟的 RPC 框架,专注业务逻辑
- 支持高并发、长连接、心跳等特性

### D3: JSON 载荷格式
- 简单直观,易于调试
- 跨语言兼容性好
- 未来可扩展为 Protobuf 提升性能

### D4: 16 字节固定头部
- 快速解析,无需读取完整载荷
- 支持流式传输与分包
