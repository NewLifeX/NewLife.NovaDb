# NewLife.NovaDb - 客户端架构

## 1. 概述

NovaDb 客户端提供标准 ADO.NET Provider 接口,支持嵌入式和服务器双模式,通过连接字符串自动识别模式。

## 2. 核心组件

### 2.1 ADO.NET Provider 类

| 类名 | 基类 | 功能 |
|------|------|------|
| `NovaClientFactory` | `DbProviderFactory` | 客户端工厂,统一创建连接/命令/参数等对象 |
| `NovaConnectionStringBuilder` | `DbConnectionStringBuilder` | 连接字符串构建与解析,支持别名映射 |
| `NovaConnection` | `DbConnection` | 连接管理,自动识别嵌入/服务器模式 |
| `NovaCommand` | `DbCommand` | SQL 执行,集成 SqlEngine |
| `NovaDataReader` | `DbDataReader` | 结果集读取 |
| `NovaParameter` | `DbParameter` | 参数化查询支持 |
| `NovaParameterCollection` | `DbParameterCollection` | 参数集合管理 |
| `NovaTransaction` | `DbTransaction` | 事务管理 |
| `NovaDataAdapter` | `DbDataAdapter` | DataSet/DataTable 数据填充 |
| `NovaBatch` | `DbBatch` | 批量命令执行（NET6.0+） |
| `NovaBatchCommand` | `DbBatchCommand` | 批量命令项（NET6.0+） |

## 3. 客户端工厂 (`NovaClientFactory`)

### 3.1 工厂模式

```csharp
public sealed class NovaClientFactory : DbProviderFactory
{
    public static readonly NovaClientFactory Instance = new();
    
    public override DbConnection CreateConnection() 
        => new NovaConnection();
    
    public override DbCommand CreateCommand() 
        => new NovaCommand();
    
    public override DbParameter CreateParameter() 
        => new NovaParameter();
    
    public override DbConnectionStringBuilder CreateConnectionStringBuilder() 
        => new NovaConnectionStringBuilder();
    
    public override DbDataAdapter CreateDataAdapter() 
        => new NovaDataAdapter();
}
```

### 3.2 工厂注册（NETSTANDARD2.1+ / NETCOREAPP）

```csharp
// 自动注册到 DbProviderFactories
DbProviderFactories.RegisterFactory("NewLife.NovaDb.Client", NovaClientFactory.Instance);

// 使用示例
var factory = DbProviderFactories.GetFactory("NewLife.NovaDb.Client");
using var conn = factory.CreateConnection();
conn.ConnectionString = "Data Source=/path/to/db";
conn.Open();
```

## 4. 连接字符串 (`NovaConnectionStringBuilder`)

### 4.1 支持的属性

| 属性 | 类型 | 默认值 | 说明 |
|------|------|-------|------|
| `DataSource` | String | - | 数据源路径（嵌入模式） |
| `Server` | String | - | 服务器地址（服务器模式） |
| `Port` | Int32 | 3306 | 端口号 |
| `Database` | String | - | 数据库名称 |
| `UserId` | String | - | 用户名（服务器模式） |
| `Password` | String | - | 密码（服务器模式） |
| `WalMode` | String | Normal | WAL 模式（嵌入模式,Full/Normal/None） |
| `ReadOnly` | Boolean | false | 是否只读模式（嵌入模式） |
| `ConnectionTimeout` | Int32 | 15 | 连接超时（秒） |
| `CommandTimeout` | Int32 | 30 | 命令超时（秒） |

### 4.2 别名支持

| 别名 | 标准属性 |
|------|---------|
| `data source`, `datasource` | DataSource |
| `server` | Server |
| `userid`, `user id`, `uid` | UserId |
| `password`, `pwd` | Password |
| `walmode`, `wal mode`, `wal` | WalMode |
| `readonly`, `read only` | ReadOnly |

### 4.3 使用示例

```csharp
// 嵌入模式
var builder = new NovaConnectionStringBuilder
{
    DataSource = "../data/member",
    WalMode = "Full",
    ReadOnly = false
};
var connStr = builder.ConnectionString;
// 输出: "Data Source=../data/member;WalMode=Full;ReadOnly=False"

// 服务器模式
builder = new NovaConnectionStringBuilder
{
    Server = "192.168.1.100",
    Port = 3306,
    Database = "member",
    UserId = "root",
    Password = "root"
};
connStr = builder.ConnectionString;
// 输出: "Server=192.168.1.100;Port=3306;Database=member;UserId=root;Password=root"
```

## 5. 连接管理 (`NovaConnection`)

### 5.1 双模式自动识别

```csharp
public class NovaConnection : DbConnection
{
    private SqlEngine? _embeddedEngine; // 嵌入模式
    private NovaClient? _remoteClient;  // 服务器模式
    
    public override void Open()
    {
        var builder = new NovaConnectionStringBuilder(ConnectionString);
        
        if (builder.IsEmbedded) // DataSource 非空
        {
            // 嵌入模式
            var options = new DbOptions
            {
                Path = builder.DataSource!,
                WalMode = ParseWalMode(builder.WalMode),
                ReadOnly = builder.ReadOnly
            };
            
            _embeddedEngine = new SqlEngine(options);
        }
        else
        {
            // 服务器模式
            _remoteClient = new NovaClient
            {
                Server = builder.Server!,
                Port = builder.Port,
                Database = builder.Database!,
                UserId = builder.UserId,
                Password = builder.Password
            };
            
            _remoteClient.Connect();
        }
        
        State = ConnectionState.Open;
    }
}
```

### 5.2 连接模式示例

```csharp
// 嵌入模式
using var conn = new NovaConnection("Data Source=../data/member");
conn.Open();

// 嵌入只读
using var conn = new NovaConnection("Data Source=../data/member;ReadOnly=true");
conn.Open();

// 嵌入完整
using var conn = new NovaConnection("Data Source=../data/member;WalMode=Full;ReadOnly=false");
conn.Open();

// 服务器模式
using var conn = new NovaConnection("Server=host;Port=3306;Database=member;UserId=root;Password=root");
conn.Open();
```

## 6. 命令执行 (`NovaCommand`)

### 6.1 非查询 SQL

```csharp
using var conn = new NovaConnection(connStr);
conn.Open();

using var cmd = conn.CreateCommand();
cmd.CommandText = "INSERT INTO users (id, name, age) VALUES (1, 'Alice', 25)";
var affectedRows = cmd.ExecuteNonQuery();
Console.WriteLine($"影响行数: {affectedRows}");
```

### 6.2 查询 SQL

```csharp
using var cmd = conn.CreateCommand();
cmd.CommandText = "SELECT * FROM users WHERE age > 18";

using var reader = cmd.ExecuteReader();
while (reader.Read())
{
    var id = reader.GetInt32(0);
    var name = reader.GetString(1);
    var age = reader.GetInt32(2);
    Console.WriteLine($"{id}, {name}, {age}");
}
```

### 6.3 参数化查询

```csharp
using var cmd = conn.CreateCommand();
cmd.CommandText = "SELECT * FROM users WHERE age > @minAge AND name LIKE @pattern";

cmd.Parameters.Add(new NovaParameter("@minAge", 18));
cmd.Parameters.Add(new NovaParameter("@pattern", "A%"));

using var reader = cmd.ExecuteReader();
while (reader.Read())
{
    Console.WriteLine(reader.GetString("name"));
}
```

### 6.4 标量查询

```csharp
using var cmd = conn.CreateCommand();
cmd.CommandText = "SELECT COUNT(*) FROM users";

var count = Convert.ToInt32(cmd.ExecuteScalar());
Console.WriteLine($"总用户数: {count}");
```

## 7. 事务管理 (`NovaTransaction`)

### 7.1 显式事务

```csharp
using var conn = new NovaConnection(connStr);
conn.Open();

using var tx = conn.BeginTransaction();

try
{
    using var cmd1 = conn.CreateCommand();
    cmd1.Transaction = tx;
    cmd1.CommandText = "UPDATE accounts SET balance = balance - 100 WHERE id = 1";
    cmd1.ExecuteNonQuery();
    
    using var cmd2 = conn.CreateCommand();
    cmd2.Transaction = tx;
    cmd2.CommandText = "UPDATE accounts SET balance = balance + 100 WHERE id = 2";
    cmd2.ExecuteNonQuery();
    
    tx.Commit();
}
catch
{
    tx.Rollback();
    throw;
}
```

### 7.2 隐式事务（未来扩展）

```csharp
// 自动提交模式（默认）
using var cmd = conn.CreateCommand();
cmd.CommandText = "INSERT INTO users (id, name) VALUES (1, 'Alice')";
cmd.ExecuteNonQuery(); // 自动提交
```

## 8. 数据适配器 (`NovaDataAdapter`)

### 8.1 填充 DataTable

```csharp

using var adapter = new NovaDataAdapter("SELECT * FROM users", conn);

var dataTable = new DataTable();
adapter.Fill(dataTable);

foreach (DataRow row in dataTable.Rows)
{
    Console.WriteLine($"{row["id"]}, {row["name"]}, {row["age"]}");
}
```

### 8.2 填充 DataSet

```csharp

var dataSet = new DataSet();
adapter.Fill(dataSet, "users");

var userTable = dataSet.Tables["users"];
```

## 10. 远程客户端 (`NovaClient`)

### 10.1 RPC 接口

```csharp
public class NovaClient : IDisposable
{
    private readonly ApiClient _client;
    
    public async Task<DateTime> PingAsync()
    {
        return await _client.InvokeAsync<DateTime>("Nova/Ping");
    }
    
    public async Task<Int32> ExecuteAsync(String sql, 
        Dictionary<String, Object?>? parameters = null)
    {
        return await _client.InvokeAsync<Int32>("Nova/Execute", 
            new { sql, parameters });
    }
    
    public async Task<QueryResult> QueryAsync(String sql, 
        Dictionary<String, Object?>? parameters = null)
    {
        return await _client.InvokeAsync<QueryResult>("Nova/Query", 
            new { sql, parameters });
    }
    
    public async Task<Int64> BeginTransactionAsync()
    {
        return await _client.InvokeAsync<Int64>("Nova/BeginTransaction");
    }
    
    public async Task CommitTransactionAsync(Int64 txId)
    {
        await _client.InvokeAsync("Nova/CommitTransaction", new { txId });
    }
    
    public async Task RollbackTransactionAsync(Int64 txId)
    {
        await _client.InvokeAsync("Nova/RollbackTransaction", new { txId });
    }
}
```

## 11. 设计决策

### D1: 标准 ADO.NET Provider
- 遵循 ADO.NET 规范,与现有 .NET 生态无缝集成
- 支持 Entity Framework、Dapper 等 ORM（未来）

### D2: 双模式自动识别
- 通过连接字符串自动识别嵌入/服务器模式
- 无需应用层修改代码,切换模式只需改连接字符串

### D3: 参数化查询
- 防 SQL 注入
- 支持查询计划复用（服务器模式）

### D4: 异步 API
- NovaClient 提供异步方法（PingAsync/ExecuteAsync/QueryAsync）
- 适合高并发服务器场景
